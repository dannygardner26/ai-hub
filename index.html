AI Workflows

4 active pipelines

Vertex AI Image Generation

Production-grade image generation using Google's Imagen 4 model and video creation with Veo 2 — all authenticated through your existing gcloud CLI. No separate API keys needed.

How it works: Prompt → curl to Vertex AI endpoint → base64 response → decoded PNG saved locally. Supports 1:1, 16:9, 9:16 aspect ratios and batch generation.
Active

Gemini Multi-Model Pipeline

Intelligent image generation with automatic fallback across 4 Gemini models. Starts with gemini-3-pro for highest quality, falls back to gemini-2.5-flash on 503 errors. Handles dark UI backgrounds with blend modes.

Model ladder: Pro (4K, best quality) → Flash 3.1 (4K, fast) → Flash 2.5 (reliable) → Flash 2.0 (cheapest). Each costs ~$0.07/image or less.
Active

AI Video Generation

Create scroll-stopping product videos by generating start and end frames with Imagen, then animating the transition. Supports deconstruction, assembly, and before/after transformations.

Pipeline: Image Generator skill creates 2 frames → feed to Runway Gen-3 ($0.50/clip), Kling 2.0 ($0.14/clip), or Veo 2 (~$1.40/4s) → 5-6s video output at 16:9.
Experimental

Prompt-to-Production Frontend

Full pipeline from a single prompt to a deployed, animated website. Combines Claude Code skills for design, Vertex AI for asset generation, and Vercel for instant deployment.

Flow: Describe the site → frontend-design skill builds it → Imagen generates visuals → 3d-animation-creator adds scroll effects → vercel --prod ships it live.
Active
